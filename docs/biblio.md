# Biblio YAML Specification

```yml
--                                                                                          
documents:                                                                                  
  - code:
    title:                                                                                  
    author:                                                                                 
    blurb:                                                                                  
    cover:                                                                                  
    buy_link:                                                                                   
    borrow_link:
    collection:    
```

## Buy Link

Add a link to the document or collection on Amazon.

## Borrow Link

Add a link to the document or collection on [WorldCat](https://www.worldcat.org/).
